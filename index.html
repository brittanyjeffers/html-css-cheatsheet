HTML & CSS Cheatsheet

A reference for some common topics in HTML and CSS

HTML...

...stands for HyperText Markup Language.

HTML elements are pieces of content consisting of an opening tag, the content, and a closing tag. Elements can display content to the viewed and provide structure to the page.

Tags provide information about the type of element being used and the information that it contains. The syntax for an opening tag is an opening angle bracket < followed by the name of the element and a closing angle bracket >. For example, <html> creates an opening html tag. The syntax for a closing tag is the same, but there is a forward slash / before the element name in the tag. For example, </html> creates a closing html tag.
